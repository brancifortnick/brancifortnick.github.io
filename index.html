Projects

Find Trails

A Goodreads clone

First full-stack group project called good trails was written in Pug and Express and changed to find-trails.com recently. The application is now using Python and React. Find trails gives the user the ability to find trails per state, and select trails they've visited, or intend on visiting, while having the option to view their personal collections. Find Trails also incorporates a rating system and description for these trails, so that the user can choose hiking trails, based on their skill level, and desired destination. Users also have the ability to write reviews on the trails.

RobinCould

Robin Could

RobinCould was a group project in week 18 where we decided to clone robinhood. The API our team used is no longer functional, so unfortunately there is no website. You may find more information, and some images from the application on my Github. Users have the ability to purchase and sell stocks, add and remove stocks from their watchlist, and review key market information. The site used ChartJS for visualizing the markets current data trends, and contained a search component that used regex for searching for the users preferred stocks.

Medley Storage

Medley

For the first time during this solo full-stack project I used AWS s3 buckets for song and image storage along with trying out a flask backend , react front end and of course a redux store. Users can upload their favorite artists and audio files along with profile images, edit, delete, and create biographies about the artist, and browse the community. The user has the ability to play music directly from the site in comparison to my first project 'Songs Smarter' where I chose to redirect the user off site to listen.

In Room Dining

In Room Dining

In-Room Dining is a web application developed to streamline employee onboarding by providing structured access to menu data, shift protocols, workflow documentation, and duty requirements. The application features a front-end–focused architecture, with a Flask backend implemented solely for user authentication processes, including secure sign-up and login endpoints. Due to corporate guidelines the site is no longer accessible, but will be rolling out another version in the near future.
