Medley

Medley

For the first time during this solo full-stack project I used AWS s3 buckets for song and image storage along with trying out a flask backend , react front end and of course a redux store. Users can upload their favorite artists and audio files along with profile images, edit, delete, and create biographies about the artist, and browse the community. The user has the ability to play music directly from the site in comparison to my first project 'Songs Smarter' where I chose to redirect the user off site to listen.
