Projects

Find Trails

A Trail Discovery Platform

Originally developed as a full-stack group project called Good Trails using Pug and Express, this application was reimagined as a solo project and rebuilt with Python and React. Find Trails enables users to discover hiking trails by state, track trails they've visited or plan to visit, and organize them into personal collections. The platform features a comprehensive rating system and detailed trail descriptions, allowing users to select hikes based on skill level and desired destination. Users can also share their experiences by writing and reading trail reviews.

Medley Storage

A Music Streaming Community

A solo full-stack project built with Flask, React, and Redux, featuring AWS S3 integration for secure song and image storage. Users can upload their favorite artists along with audio files and profile images, create and edit artist biographies, and explore content shared by the community. Unlike previous projects, Medley Storage allows users to stream music directly on the platform, providing a seamless listening experience without external redirects.

LocateSteaks

A Cheesesteak Rating Platform

Built with React, Redux, Flask, PostgreSQL, and Docker, LocateSteaks is a community-driven platform for discovering and rating cheesesteak establishments. The application features a voting system that allows users to rate their favorite spots and share recommendations. While core functionality is live and deployed on Heroku, future enhancements will include interactive maps integration for location-based searches and improved address visualization.

Songs Smarter

A Music Sharing Community

My first full-stack solo project, Songs Smarter is a community platform where musicians can upload their original music, share it with other users, and receive feedback through comments. This project served as a foundation for exploring full-stack development concepts. Future enhancements include implementing a marketplace feature that will allow users to buy and sell melody stems, creating opportunities for collaboration and monetization within the creative community.

Robinhood Clone

A Trading Platform Clone

A full-stack web application that replicates the core functionality of Robinhood, allowing users to trade stocks, manage portfolios, and track market performance in real-time. Built with React, Redux, Flask, and PostgreSQL, this application features user authentication, watchlists, portfolio management, and live stock data integration. Please note that due to API rate limiting on the free tier, some stock data may be cached or limited to ensure optimal performance and availability for all users.

In Room Dining

An Employee Onboarding Platform

A specialized web application designed to streamline employee onboarding in the hospitality industry. The platform provides structured access to menu data, shift protocols, workflow documentation, and duty requirements. Built with a front-end-focused architecture and a Flask backend for secure authentication, the application simplifies training and operational efficiency. Please note that due to corporate guidelines, the site is currently unavailable, but a new version will be released in the near future.
